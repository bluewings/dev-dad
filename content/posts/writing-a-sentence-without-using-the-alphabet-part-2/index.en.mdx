## Master Key üîë

You can use the [unescape()](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/unescape) function as a master key to generate any character.

> The `unescape()` function computes a new string in which hexadecimal escape sequences are replaced with the character that it represents.

```javascript{7-8}
/**
 * @param {string} str - A string to be decoded.
 * @return {string} A new string in which certain characters have been unescaped.
 * @example unescape(str)
 */

unescape('abc123');         // "abc123"
unescape('%uC548%uB155');   // "ÏïàÎÖï"
unescape('%uD83D%uDC4B');   // "üëã"
```

As shown in the example above, you can create any character you want with the `unescape` function and the `% + HEXA_ASCII_VALUE` value. But to execute that function, two things must be solved.

1. Convert string to executable script

2. Two characters <code>p</code> and <code>%</code> are required to create the script <code>unesca<span style="color:red">p</span>e('<span style="color:red">%</span>41')</code><br/>The remaining characters can be obtained by the method described in the <a href="/en/writing-a-sentence-without-using-the-alphabet/">previous post</a>.

## 1. Convert string to executable script

The [Function](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Function) constructor lets you create an executable function.

> The __`Function` constructor__ creates a new `Function` __object__. Calling the constructor directly can create functions dynamically.


```javascript{8}
/**
 * @param {string} arg1, arg1, ...argN
 * @param {string} functionBody
 * @return {function} executable function.
 * @example var sum = Function('a', 'b', 'return a + b');
 */

Function('return unescape')     // ∆í anonymous() { return unescape }
Function('return unescape')()   // ∆í unescape() { [native code] }
```

The Function constructor can be obtained in the following way.

```js
({}).constructor.constructor    // ∆í Function() { [native code] }
```
